## üñºÔ∏è Logo Guidelines

### Logo Concepts

#### Option 1: Minimalist Truck Icon
- Clean geometric moving truck
- Simple, recognizable shape
- Works in single color or full color
- Scalable from favicon to billboard

#### Option 2: Badge/Shield Design
- Circular or shield-shaped badge
- Truck silhouette in center
- Company name around perimeter
- Conveys trust and protection

#### Option 3: Text + Icon
- Company name in bold sans-serif
- Truck icon to the left
- Clean, professional layout
- Easy to read at any size

### Logo Usage Rules
1. **Minimum Size**: 40px height (digital), 0.5 inches (print)
2. **Clear Space**: Minimum padding equal to the height of the letter 'J'
3. **Background**: Should work on white, light backgrounds, and dark backgrounds
4. **Variations Needed**:
   - Full color version
   - White version (for dark backgrounds)
   - Single color version
   - Favicon/app icon version (square)

## üé¨ Tone of Voice

### Personality Traits
- **Professional** but not stuffy
- **Friendly** but not casual
- **Confident** but not arrogant
- **Helpful** but not pushy
- **Local** (Toronto pride)

### Writing Style
- Use active voice
- Short, clear sentences
- Avoid jargon
- Address customer pain points
- Focus on solutions

### Example Phrases
- ‚úÖ "We handle every detail with care"
- ‚úÖ "Your move, simplified"
- ‚úÖ "Toronto's trusted moving partner"
- ‚úÖ "Stress-free relocations, guaranteed"
